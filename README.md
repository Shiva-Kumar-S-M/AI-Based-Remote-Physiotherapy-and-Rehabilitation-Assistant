AI Remote Physiotherapy Assistant Because recovery shouldn’t depend on where you live.

🌍 Why I Built This
Physiotherapy is powerful — but not always accessible.
Many people recovering from injuries, surgeries, or chronic pain don’t have regular access to professional supervision. Sometimes it’s too expensive. Sometimes the nearest clinic is too far. Sometimes life just gets in the way.
And when exercises are done incorrectly, recovery becomes slower… or worse, painful again.
This project is my attempt to solve that.

💡 What This Project Does

This is a browser-based AI rehabilitation assistant that helps users perform physiotherapy exercises safely at home.

It:
Watches posture using real-time pose detection
Counts repetitions automatically
Tracks progress
Provides instant movement feedback
Suggests exercises (especially senior-friendly routines)
Runs directly in the browser using simple web technologies
No heavy setup. No expensive equipment. Just a device with a camera.

🛠 Built With Simplicity in Mind
HTML
CSS
JavaScript
Browser-based pose estimation
I intentionally kept it lightweight so it can run on low-cost devices and be accessible to more people.

👥 Who This Can Help
Patients recovering from injury
Senior citizens needing guided exercises
People in rural or remote areas
Anyone wanting structured, safe home workouts
Preventive healthcare users

❤️ The Bigger Vision
Healthcare shouldn’t be limited by geography or affordability.
This project is not about replacing physiotherapists.
It’s about extending their reach.
If AI can help even one person recover more confidently at home, then it’s worth building.
